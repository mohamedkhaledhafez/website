Review and breakdown of The Web Developer Bootcamp 2020

What is The Web Developer Bootcamp 2020?#

This is full-stack bootcamp taught by Colt Steele, covering all major areas of web development including front end, backend, databases and version control. I have personally taken this course hosted on udemy. This course is very comprehensive it has also just been updated as of October 2020! Look below for some quick points of the The Web Developer Bootcamp 2020

Quick facts#

  • Fully online 100%
  • Updated October 2020
  • Massive online community Facebook group 27.0K members
  • Covers full development circle from build to production
  • Usually can be bought for \$10

Whats covered in the course?#

  • HTML5
  • CSS3
  • Flexbox, Responsive Design
  • JavaScript (all 2020 modern syntax, ES6, ES2018, etc.)
  • Asynchronous JavaScript - Promises, async/await, etc.
  • AJAX and single page apps
  • Bootstrap 4 and 5 (alpha)
  • SemanticUI
  • DOM Manipulation
  • Unix(Command Line) Commands
  • NodeJS
  • ExpressJS
  • NPM
  • Mongoose
  • Database Associations
  • Deploying Apps
  • Cookies & Sessions
  • Much more!!

Should I buy?#

There are many free resources for learning to code such as Freecodecamp But if your looking to hit the ground running and want all the content in order then this at the price is the best investment you can make.

Why pay hundreds of dollars when this is availble?
